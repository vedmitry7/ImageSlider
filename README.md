Тестовое задание для позиции
Android developer

Выполненные требования:<br/>

Junior<br/>
1) выбор источника данных - папка во внутренней памяти или SD-карте<br/>
2) реализован запуск слайд-шоу в заданное пользователем время<br/>
3) реализована остановк слайд-шоу в заданное пользователем
время<br/>
4) реализована возможность задавать длину интервала между
кадрами (в секундах, от 1 до 60)<br/>
5) реализован автоматический запуск во время зарядки устройства<br/>
6) реализован автоматический запуск после перезагрузки
устройства<br/>

Middle<br/>
1) реализована возможность сохранять все настройки
приложения в памяти устройства (что происходит автоматически)
