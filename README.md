Тестовое задание для позиции
Android developer

Выполненные требования:

Junior<br/>
1) выбор источника данных - папка во внутренней памяти или SD-карте<br/>
2) необходимо реализовать запуск слайд-шоу в заданное пользователем время<br/>
3) необходимо реализовать остановку слайд-шоу в заданное пользователем
время
4) у пользователя должна быть возможность задавать длину интервала между
кадрами (в секундах, от 1 до 60)
5) необходимо реализовать автоматический запуск во время зарядки устройства
6) необходимо реализовать автоматический запуск после перезагрузки
устройства

Middle
1) у пользователя должна быть возможность сохранять все настройки
приложения в памяти устройства
